Sync your Wordpress user accounts with Salesforce.

== Description ==

With User Sync for Salesforce you can add your WordPress users to your contacts in Salesforce. User Sync for Salesforce also supports WooCommerce, WooCommerce address fields will automatically be added to your Salesforce contacts.

Note: this plugin is in beta stage, chances are that you will encounter bugs. Please let me know when you do, they will be fixed.

== Installation ==

= Installation =
1. Upload `salesforce-user-sync` to the `/wp-content/plugins/` directory,
2. Activate the plugin through the 'Plugins' menu in WordPress
